2. 일반적인 프로그램은 Windows 나  Linux 같은 운영체제(Operating System) 위에서 실행 되므로
윈도우즈에서 동작하도록 구현된 워드 프로그램은 리눅스 기반에서 동작하지 않는다.
리눅스에서 동작시키려면 리눅스를 기반으로 다시 구현해야 한다.

이와 비교하여 자바 프로그램의 실행구조를 설명하시오.
-운영체제는 자바 가상머신을 실행시키고 자바 가상머신은 자바 프로그램을 실행시키는 구조

3. 자바가상머신(JVM) 이란?
-자바 소스코드를 컴파일하게 되면 바이트 코드로 변환되는데, JVM은 이러한 바이트 코드를 읽어서 해당 플랫폼에서 이해할 수 있는 형태로 해석하여 프로그램을 실행시켜 줌

8. 자바 소스코드가 컴파일되고, 실행되는 과정을 자세히 설명하시오.
자바 소스코드를 작성 후, javac를 통해 컴파일을 진행 합니다
이후, 컴파일이 완료되면 java.exe 를 실행하여 JVM을 작동하여 바이트코드를 읽고 자바를 실행하게 됩니다.
